Shoppiko is a high-performance, full-stack e-commerce application built using the MERN stack. It features a complete shopping experience from product discovery to secure checkout, including a dedicated Admin Dashboard for inventory management.

 Key Features

- **User Authentication:** Secure Login/Signup using **JWT (JSON Web Tokens)** and encrypted passwords.
- **Product Management:** Full CRUD operations for products (Admin only).
- **Shopping Cart:** Persistent cart functionality allowing users to add/remove items and adjust quantities.
- **Search & Filter:** Real-time product search and category filtering for a seamless user experience.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop using **Tailwind CSS**.
- **Admin Dashboard:** Private route for admins to track orders and manage stock.
